# Support

## About This Fork

This is a **custom fork** of [microsoft/dbt-fabricspark](https://github.com/microsoft/dbt-fabricspark) maintained by **Plainsight**. This fork was created because the original repository is not being actively maintained, and we needed a working solution for our projects.

## How to Get Help

### For Issues Specific to This Fork

This project uses GitHub Issues to track bugs and feature requests. Please search the existing issues before filing new ones to avoid duplicates.

- **Report a bug or issue:** Open a new issue at [github.com/PlainsightPro/dbt-fabricspark-plainsight/issues](https://github.com/PlainsightPro/dbt-fabricspark-plainsight/issues/new)
- **Request a feature:** Open a new issue with the "enhancement" label
- **Ask questions:** Use the issue tracker for questions specific to this fork

Please provide as much detail as possible when filing issues:
- Steps to reproduce the problem
- Expected vs. actual behavior
- Environment details (OS, Python version, dbt version)
- Relevant logs or error messages

### For General dbt Questions

For questions about dbt in general (not specific to this fork):
- Join the [dbt Community Slack](http://community.getdbt.com/)
- Check the [dbt Community Discourse](https://discourse.getdbt.com)
- Review the [dbt Documentation](https://docs.getdbt.com/)

### For Issues with the Original Adapter

For issues related to the upstream Microsoft Fabric Spark adapter:
- Visit the [microsoft/dbt-fabricspark repository](https://github.com/microsoft/dbt-fabricspark)

## Support Policy

Support for this **custom fork** is provided on a best-effort basis by Plainsight. This fork is maintained primarily for Plainsight's internal projects, but we welcome community contributions and issue reports.

**Note:** This is not an officially supported Microsoft product. For official Microsoft support, please refer to the original [microsoft/dbt-fabricspark](https://github.com/microsoft/dbt-fabricspark) repository.
